Install Node.js 4.x

curl -sL https://deb.nodesource.com/setup_4.x | sudo -E bash - sudo apt-get install -y nodejs

Install Node.js 5.x

curl -sL https://deb.nodesource.com/setup_5.x | sudo -E bash - sudo apt-get install -y nodejs

Optional build essential to compile npm

sudo apt-get install -y build-essential

Mozilla Firefox 32 Bit

cd /usr/local wget http://ftp.mozilla.org/pub/mozilla.org/firefox/releases/45.0/linux-i686/en-US/firefox-45.0.tar.bz2

Mozilla Firefox 64 Bit

cd /usr/local wget http://ftp.mozilla.org/pub/mozilla.org/firefox/releases/45.0/linux-x86_64/en-US/firefox-45.0.tar.bz2
